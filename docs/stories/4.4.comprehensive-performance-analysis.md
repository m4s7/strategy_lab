## Dev Notes
This story creates the comprehensive performance analysis system that provides deep insights into strategy behavior and performance characteristics.

**Performance Analysis Architecture:**
- **Core Module**: src/strategy_lab/analysis/
- **Key Components**: performance/, risk/, trade/, visualization/
- **Integration**: Works with backtesting results and optimization outputs
- **Output**: Detailed reports, charts, and statistical analysis

**Comprehensive Metrics Suite:**
- **Profitability**: Total return, CAGR, profit factor, win rate
- **Risk**: Volatility, Sharpe ratio, Sortino ratio, Calmar ratio
- **Drawdown**: Maximum drawdown, average drawdown, recovery time
- **Advanced Risk**: VaR, CVaR, tail ratio, skewness, kurtosis
- **Trade Statistics**: Average trade, hit rate, profit/loss ratio

**Time-series Analysis Features:**
- **Rolling Metrics**: Moving window performance calculations
- **Regime Detection**: Identify performance regime changes
- **Seasonality**: Monthly, daily, hourly performance patterns
- **Attribution**: Performance attribution to market conditions
- **Stability**: Performance consistency and reliability measures

**Trade Analysis Components:**
- **Individual Trades**: Entry/exit analysis, holding periods, P&L distribution
- **Trade Clustering**: Identify consecutive wins/losses patterns
- **Timing Analysis**: Entry/exit timing quality and slippage
- **Position Sizing**: Size effectiveness and risk contribution
- **Market Conditions**: Trade performance across different market states
