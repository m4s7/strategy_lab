# Story 5.3: Performance Metrics Calculation

## Status
Ready for Review

## Story
**As a** quantitative trader using Strategy Lab,
**I want** comprehensive performance metrics calculated and displayed after backtests,
**so that** I can evaluate my strategy's risk-adjusted returns and make informed trading decisions.

## Acceptance Criteria

1. **Return Metrics**: Calculate total return, annualized return, and excess return over risk-free rate
2. **Risk Metrics**: Compute Sharpe ratio, Sortino ratio, maximum drawdown, and volatility measurements
3. **Trade Analysis**: Generate win rate, profit factor, average trade P&L, and trade distribution statistics
4. **Risk Management**: Calculate Value at Risk (VaR), Conditional VaR, and risk-adjusted metrics
5. **Time Series**: Produce equity curve, drawdown series, and rolling performance windows
6. **Display Integration**: Replace CLI placeholder results with actual calculated metrics
7. **Export Capability**: Save detailed results to files when output directory is specified
8. **Validation**: Ensure all calculations are mathematically correct and industry-standard

### Standard Performance Metrics
- **Sharpe Ratio**: (Portfolio Return - Risk Free Rate) / Portfolio Volatility
- **Sortino Ratio**: (Portfolio Return - Risk Free Rate) / Downside Deviation
- **Maximum Drawdown**: Maximum peak-to-trough decline in equity
- **Win Rate**: Number of profitable trades / Total number of trades
- **Profit Factor**: Gross Profit / Gross Loss
- **VaR (95%)**: 5th percentile of return distribution

- Summary metrics in tabular format

- Clear formatting with proper decimal precision
